Selecting Elements

getElementById

Returns the Element as an object or null (if not found).

getElementByClassName

Returns the Elements as an HTML Collection (arrays of objects) or empty collection(if not found).

getElementByTagName

Returns the Elements as an HTML Collection (arrays of objects) or empty collection(if not found).

Query Selectors

Allows us to use any CSS selector

document.querySelector('p'); //Selects first p element
document.querySelector('myId'); //Selects first element with id = myId
document.querySelector('.myclass'); //Selects first element with class = myClass
document.querySelector('.className:nth-of-type(n+1)'); //Selects alternative element with class = myClass
document.querySelectorAll("p"); //Selects all p elements
