Arrays Methods

1.Push : add to end (add that element in the array end)

2.Pop : delete from end & returns it

3.Unshift : add to start (add that element in the array start)

4.Shift : delete from start & returns it

5.indexOf : returns index of something (index of the element and if not present then -1 will be output)

6.includes : search for value (Tells that value is present i.e. true and if not i.e. false)

7.concat : merge 2 arrays . It does not changes in original array.

8.reverse : reverse an array. It changes in original array.

9.slice : copies a portion of an array.It does not changes in original array.

10.splice : removes / replaces / add elements in place. splice(start,deleteCount,item0...itemN).It changes in original array.

11.sort : sorts an array
